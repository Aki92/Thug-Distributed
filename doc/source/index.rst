Welcome to Thug Distributed's documentation!
============================================

`ThugD <https://github.com/Aki92/Thug-Distributed>`_ is short form of **Thug Distributed Task Queuing Project**. It is undergoing as a GSoC Project under The Honeynet Project organization. It will work as an enhancement to the existing `Thug Project <http://buffer.github.io/thug/>`_.

Please refer to `Project Slot Page <https://www.honeynet.org/gsoc/slot3>`_ for more details.

Stay tuned to `Project Weekly Blog <http://gsoc2013.honeynet.org/category/project-3-thug-distributed-task-queuing>`_.
